kkk 近邻学习

kkk 近邻（kNNk\text{NN}kNN）学习是一种常用的监督学习方法，其基本步骤可归纳如下。

  1. 确定训练样本与某种距离度量。
  2. 对于某个给定的测试样本，找到训练集中距离最近的 kkk 个样本。
  3. 在分类问题中使用“投票法”获得预测结果，而在回归问题中使用“平均法”获得预测结果，甚至可以基于距离远近进行加权投票或加权平均，其中距离越近的样本权重越大。
    • 投票法：选择 kkk 个样本中出现最多的类别标记作为预测结果。
    • 平均法：将 kkk 个样本的实值输出标记的平均值作为预测结果。

kNNk\text{NN}kNN 没有显式的训练过程，属于懒惰学习。所谓懒惰学习，是指此类学习在训练阶段仅保存样本，训练时间开销为零，待收到测试样本后再进行处理。对应地，急切学习是指此类学习在训练阶段就对样本进行学习处理的方法。

在 kNNk\text{NN}kNN 分类器中，当 kkk 取不同值时，分类结果会有显著不同；此外，若采用不同的距离计算方式，则找到的近邻样本可能也有显著区别，从而导致分类结果有显著不同。

设距离计算能恰当地找出 kkk 个近邻，下面对最近邻分类器（1NN1\text{NN}1NN）在二分类问题上的性能进行讨论。

给定测试样本 x\bm{x}x，若其最近邻样本为 z\bm{z}z，则 1NN1\text{NN}1NN 出错的概率就是 x\bm{x}x 和 z\bm{z}z 类别标记不同的概率，即

P(err)=1−∑c∈YP( c ∣ x )P( c ∣ z )P(err)=1-\sum\limits_{c\in\mathcal{Y} }P(\,c\,|\,\bm{x}\,)P(\,c\,|\,\bm{z}\,) P(err)=1−c∈Y∑​P(c∣x)P(c∣z)

假设样本独立同分布，且对于任意 x\bm{x}x 和任意小的正整数 δ\deltaδ，在 x\bm{x}x 附近的 δ\deltaδ 距离范围内总能找到一个训练样本。换而言之，对任意测试样本，我们总能在任意近的范围内找到上式中的训练样本 z\bm{z}z。令 c∗=arg max⁡c∈YP( c ∣ x )c^*=\argmax\limits_{c\in\mathcal{Y}}P(\,c\,|\,\bm{x}\,)c∗=c∈Yargmax​P(c∣x) 表示贝叶斯最优分类器的结果，有

P(err)=1−∑c∈YP( c ∣ x )P( c ∣ z )≃1−∑c∈YP2( c ∣ x )⩽1−P2( c∗ ∣ x )=(1+P( c∗ ∣ x ))(1−P( c∗ ∣ x ))⩽2×(1−P( c∗ ∣ x ))\begin{aligned} P(err)&=1-\sum\limits_{c\in\mathcal{Y} }P(\,c\,|\,\bm{x}\,)P(\,c\,|\,\bm{z}\,)\\ &\simeq 1-\sum\limits_{c\in\mathcal{Y} }P^2(\,c\,|\,\bm{x}\,)\\ &\leqslant 1-P^2(\,c^*\,|\,\bm{x}\,)=(1+P(\,c^*\,|\,\bm{x}\,))(1-P(\,c^*\,|\,\bm{x}\,))\\ &\leqslant 2\times(1-P(\,c^*\,|\,\bm{x}\,)) \end{aligned} P(err)​=1−c∈Y∑​P(c∣x)P(c∣z)≃1−c∈Y∑​P2(c∣x)⩽1−P2(c∗∣x)=(1+P(c∗∣x))(1−P(c∗∣x))⩽2×(1−P(c∗∣x))​

由此我们得出结论：1NN1\text{NN}1NN 的泛化错误率不超过贝叶斯最优分类器错误率的两倍。

低维嵌入

上述讨论基于一个重要的假设：任意测试样本 x\bm{x}x 附近的任意小的 δ\deltaδ 距离范围内总能找到一个训练样本，即训练样本的采样密度足够大，或称为“密采样”。然而，该假设在现实任务中通常很难满足。事实上，在高维情形下出现的数据样本稀疏、距离计算困难等问题是所有机器学习方法共同面临的严重障碍，其被称为维数灾难。
