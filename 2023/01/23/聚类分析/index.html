聚类任务

相比于监督学习，无监督学习的训练样本集不带有标签。一般地，在无监督学习任务中研究最多、应用最广的是聚类任务，其目标是将数据集中的样本划分为若干个通常不相交的子集，称为簇。聚类既可以作为一个单独过程以寻找数据内在的分布结构，也可以作为作为分类等其他学习任务的前驱过程。

聚类任务的形式化描述如下。

  1. 设样本集 D={x1,⋯ ,xm}D=\{\bm{x}_1,\cdots,\bm{x}_m\}D={x1​,⋯,xm​} 包含 mmm 个无标记样本，其中每个样本 xi=(xi1,⋯ ,xin)\bm{x}_i=(x_{i1},\cdots,x_{in})xi​=(xi1​,⋯,xin​) 均为 nnn 维特征向量。
  2. 将 DDD 划分成 kkk 个不相交的簇 { Cl ∣ l∈[ 1,k ] }\{\,C_l\,|\,l\in[\,1,k\,]\,\}{Cl​∣l∈[1,k]}，其中 Cl′∩l′≠lCl=∅C_{l'}\mathop{\cap}\limits_{l'\neq l}C_l=\varnothingCl′​l′=l∩​Cl​=∅ 且 D=⋃l=1kClD=\bigcup\limits_{l=1}^kC_lD=l=1⋃k​Cl​。
  3. 用 λj∈{1,2,⋯ ,k}\lambda_j\in\{1,2,\cdots,k\}λj​∈{1,2,⋯,k} 表示样本的簇标记，即 xj∈Cλj\bm{x}_j\in C_{\lambda_j}xj​∈Cλj​​。
  4. 聚类的结果可用包含 mmm 个元素的簇标记向量 λ=(λ1,λ2,⋯ ,λm)\bm{\lambda}=(\lambda _1,\lambda _2,\cdots,\lambda _m)λ=(λ1​,λ2​,⋯,λm​) 表示。

原型聚类

原型聚类也称为“基于原型的聚类”，此类算法假设聚类结构能通过一组原型刻画。通常情况下，算法先对原型进行初始化，再对原型进行迭代更新求解。

密度聚类

密度聚类也称为“基于密度的聚类”，此类算法假设聚类结构能通过样本分布的紧密程度以确定。通常情况下，密度聚类算法从样本密度的角度考察样本之间的可连接性，并基于可连接样本不断扩展聚类簇来获得最终的聚类结果。

下面我们将介绍密度聚类中的 DBSCAN 算法，其由给定的一组邻域参数 (ϵ,MinPts)(\epsilon,\it{MinPts})(ϵ,MinPts) 来刻画样本分布的紧密程度。

一些基本概念。

  1. ϵ\epsilonϵ 邻域：对样本 xj∈D\bm{x}_j\in Dxj​∈D，其 ϵ\epsilonϵ 邻域包含 DDD 中与 xj\bm{x}_jxj​ 的距离不大于 ϵ\epsilonϵ 的样本。
  2. 核心对象：若样本 xj\bm{x}_jxj​ 的 ϵ\epsilonϵ 邻域至少包含 MinPts\it{MinPts}MinPts 个样本，则该样本点为一个核心对象。
  3. 密度直达：若样本 xj\bm{x}_jxj​ 位于 xi\bm{x}_ixi​ 的 ϵ\epsilonϵ 邻域中，且 xi\bm{x}_ixi​ 为核心对象，则称 xj\bm{x}_jxj​ 由 xi\bm{x}_ixi​ 密度直达。
  4. 密度可达：对于 xi\bm{x}_ixi​ 与 xj\bm{x}_jxj​，若存在样本序列 p1=xi,p2,⋯ ,pn=xjp_1=\bm{x}_i,p_2,\cdots,p_n=\bm{x}_jp1​=xi​,p2​,⋯,pn​=xj​，其中 pi+1p_{i+1}pi+1​ 由 pip_ipi​ 密度直达，则称该两样本密度可达。

层次聚类

层次聚类在不同层次对数据集进行划分，从而形成树形的聚类结构。数据集划分既可采用“自底向上”的聚合策略，也可采用“自顶向下”的分拆策略。

下面我们将介绍层次聚类中的 AGNES 算法。

首先将样本中的每一个样本看做一个初始聚类簇，然后在算法运行的每一步中找出距离最近的两个聚类簇进行合并，该过程不断重复，直到达到预设的聚类簇的个数。

此处两个聚类簇 CiC_iCi​ 和 CjC_jCj​ 的距离有三种度量方式，如下所示。
